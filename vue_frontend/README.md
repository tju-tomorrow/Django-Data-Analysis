# LogOracle 日志神谕 - 前端客户端

智能日志分析平台的前端应用，基于 Vue 3 + Vite 构建。

## 项目简介

LogOracle（日志神谕）是一个基于 RAG（检索增强生成）技术的智能日志分析平台，能够：

- 🔍 **智能检索**：通过向量检索技术快速定位相关日志
- 💡 **深度分析**：使用大语言模型进行专业的日志分析
- 🎯 **精准诊断**：识别问题根因并提供解决方案
- 📊 **多维度洞察**：支持错误分类、性能分析、安全分析等多种模式

## 技术栈

- Vue 3 (Composition API)
- Vite
- Vue Router
- Pinia (状态管理)
- Axios (HTTP 客户端)
- Marked (Markdown 渲染)

## 开发

